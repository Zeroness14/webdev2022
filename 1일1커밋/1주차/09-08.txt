1. IntelliJ 프로젝트 생성

1) New Project 버튼을 눌러 새로운 프로젝트를 생성한다.
 - 기존에 사용하던 프로젝트가 있다면 다음 단계인 [인텔리제이 셋팅] 부분으로 넘어간다.

2) 기본적인 java 프로젝트로 진행하면 프로젝트를 생성하였던 창의 아래쪽의 톱니바퀴 아이콘을 눌러 JDK를 선택할 수 있고 
프로젝트를 생성하여 셋팅을 진행해보면, 왼쪽의 메뉴바에서 프로젝트 유형을 선택한 뒤, 오른쪽 상단의 Project SDK를 선택한다.
기존에 IntelliJ에 사용하던 JDK가 있다면 그것을 사용해도 되고, 등록이 안된 JDK를 사용하고 싶다면,
Add JDK... 을 눌러 IntelliJ에 추가하여 원하는 버전의 JDK를 선택하여 사용한다.
 
3) 원하는 버전의 JDK를 선택한 후 Next 버튼을 눌러 다음 단계로 넘어간다.

4) Next 버튼을 눌러 다음 단계로 넘어간다.

5) Project name과 Project location을 설정해 준 뒤 Finish 버튼을 눌러 마무리한다.
 
6) 기존에 없는 경로로 설정했다면, 위와 같은 알림 문구가 나타나는데
Create 버튼을 누르면 해당 경로를 IntelliJ에서 만든다.
